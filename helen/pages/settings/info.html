Introducción

Hellen es un sistema interactivo enfocado en mejorar la accesibilidad de personas sordas, proporcionando herramientas visuales como reloj, clima y alarmas mediante una interfaz amigable.

Objetivo

Brindar una plataforma accesible e intuitiva para personas con discapacidad auditiva mediante la integración de hardware, software y diseño accesible.

Tecnologías

  • HTML5, CSS3, JavaScript
  • Node.js + Express
  • Socket.IO
  • Bootstrap Icons
  • OpenWeatherMap API
  • TimeZoneDB API
  • Raspberry Pi
  • Cámara 5MP
  • Pantalla táctil
